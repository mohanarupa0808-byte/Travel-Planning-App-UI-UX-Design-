# Travel Planning App UI-UX Design - Trego📍✈️ 🏝️
Trego is a modern travel planning app UI/UX design focused on simplicity and seamless user experience. It includes destination search, trip planning, booking flow, and itinerary management with ₹ pricing. Designed using Figma with a clean, mobile-first approach optimized for usability and low data usage.
